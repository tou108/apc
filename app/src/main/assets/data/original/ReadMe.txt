頑張って作ったモンハンシミュレータ for MHXX ver.0.9
2017/3/26

■説明
ニンテンドー３ＤＳ専用ソフト「モンスターハンターダブルクロス」
のゲーム内において、発動スキルに対応する防具の組み合わせを検索する
「スキルシミュレータ」「スキルサーチ」と呼ばれるツールの一つです。

■動作環境
Java Runtime Environment Version 8　以上、動かない人はインストールが必要です。
OSは、Javaが動けば多分何でもOKだと思います。

■使い方
ダウンロードしたzipファイルを適当なフォルダに解凍して下さい。
頑シミュ.jarをダブルクリックすると起動します。
環境が良く分からない方などは下記のサイトを確認下さい。
最新版についてもこちらで配布しています。リンクフリーです。
http://www.geocities.jp/masax_mh/

旧バージョンの保存データを引き継ぐ場合、
旧バージョンのsaveフォルダを新しくダウンロードしたフォルダ内にコピーして下さい。
